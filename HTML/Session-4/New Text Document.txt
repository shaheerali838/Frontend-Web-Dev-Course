🔹 2. Introduction to HTML Tables

🔹 3. Basic Table Structure
Concepts to Cover:
• table – defines the full table
• tr – defines a row
• td – defines a standard data cell
• th – defines a header cell

Examples:
• Simple table with 2 rows and 2 columns
• Use th for headings and td for data

🔹 4. Table Attributes and Modern Styling
Concepts to Cover:
• Old HTML attribute: border="1" (deprecated)
• Modern way: use CSS styling
• Introduce padding, spacing using CSS
• border-collapse: collapse

Examples:
• Inline CSS for styling cells
• Spacing between cells using border-spacing

🔹 5. Merging Cells with Colspan and Rowspan
Concepts to Cover:
• colspan – merge columns
• rowspan – merge rows
• Where and why to use merged cells

Examples:
• Timetable or calendar layout
• Table header spanning multiple columns

🔹 6. Semantic Table Sections
Concepts to Cover:
• thead – for heading rows
• tbody – for main content
• tfoot – for summary or total rows
• Benefits of semantic structure (readability, accessibility)

Examples:
• Table divided into thead, tbody, and tfoot
• Simple total row using colspan

Concepts to Cover:

Examples:
